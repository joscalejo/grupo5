Cifrado César — Grupo 5

Interfaz web simple de nuestro proyecto, permite al usuario cifrar y descifrar manteniendo mayúsculas y minúsculas, además tiene la opcion de incluir la rotacion de números.

Explicación del Cifrado César

Esta sección explica paso a paso cómo se aplica la aritmética modular en el cifrado César. Usa los controles para probar cualquier carácter (o toma la primera letra del texto ingresado).

Se usará la primera letra del texto, pero puedes editarla aquí.

Pasos

  1. Convertir: letra → número (índice en el diccionario).
  2. Sumar: añadir el desplazamiento k.
  3. Aplicar módulo: usar (i + k) mod n para que el índice quede dentro del diccionario.
  4. Convertir: número → nuevo carácter.

Tabla de mapeo

Preserva mayúsculas. Los caracteres no definidos (puntuación, espacios) se mantienen.
